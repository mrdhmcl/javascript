Array introduction

What is an array?

Array is a type of composite variable that can store several values inside of one only variable.

Example: let number = [4, 8, 5].

Array parts

- Array: it is basically the whole thing, including the variable and its values.

- Index/key: it's the position of the values inside the array. It starts with 0. In the example we used, the index 0 is 4, index 1 is 8 and index 2 is 5.

- Content/value: it's the content inside of each index. It can be a number or a string.

- Element: it is the spot filled by every content.
